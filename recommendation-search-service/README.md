## Description

Recommendation & Search microservice (NestJS + MongoDB) for the bookshop.

- Runs on port `8083` by default
- Exposed via API Gateway at `http://localhost:8060` (see config in `bookshop-config-server`)

This service supports:

- Full‑text search over an indexed set of books
- Simple personalized recommendations based on user events (views/purchases)

## Project setup

```bash
$ pnpm install
```

## Environment

Copy `.env.example` to `.env` (optional) and adjust values:

- `PORT` (default `8083`)
- `MONGODB_URI` (default `mongodb://localhost:27017/recommendation_search`)

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Endpoints

Base URL (direct): `http://localhost:8083`

Base URL (via gateway): `http://localhost:8060`

- `GET /health`
  - Example: `curl -s http://localhost:8060/health`

- `POST /index/books` (upsert index)
  - Body:
    - `{ "books": [{ "bookId": "...", "title": "...", "author"?: "...", "description"?: "...", "categoryName"?: "...", "imageUrl"?: "...", "price"?: 12.3 }] }`
  - Example:
    - `curl -s -X POST http://localhost:8060/index/books -H 'Content-Type: application/json' -d '{"books":[{"bookId":"1","title":"Dune","author":"Frank Herbert","categoryName":"Sci-Fi","description":"Sci-fi classic"}]}'`

- `GET /search?q=...&limit=...`
  - Example:
    - `curl -s 'http://localhost:8060/search?q=dune&limit=10'`

- `POST /events` (record interaction)
  - Body:
    - `{ "userId": 1, "bookId": "1", "type": "VIEW" | "PURCHASE", "categoryName"?: "Sci-Fi" }`
  - Example:
    - `curl -s -X POST http://localhost:8060/events -H 'Content-Type: application/json' -d '{"userId":1,"bookId":"1","type":"VIEW"}'`

- `GET /recommendations/user/:userId?limit=...`
  - Example:
    - `curl -s 'http://localhost:8060/recommendations/user/1?limit=12'`
